ОПИСАНИЕ
--------
Класическата аркадна игра Pac-Man, реализирана на Python
с графична библиотека Pygame.
Играчът управлява Pac-Man, събира точки и избягва духове.
При изяждане на голяма жълта точка (Power-Up) Pac-Man може
да погълне духовете за кратко.


СТРУКТУРА НА ПРОЕКТА
--------------------
pacman/
│
├── models.py         ← класът PacManGame (логика на играта)
├── main.py           ← стартиращ файл, pygame цикъл
├── requirements.txt  ← необходими библиотеки
└── readme.txt        ← това описание


ИЗПОЛЗВАНИ БИБЛИОТЕКИ
---------------------
  pygame 2.6.1  – графика, звук, управление (pip install pygame)


ИНСТАЛАЦИЯ
----------
1. Уверете се, че имате Python 3.9+:
       python --version

2. Инсталирайте зависимостите:
       pip install -r requirements.txt

3. Стартирайте играта:
       python main.py


УПРАВЛЕНИЕ
----------
  ↑ / W  – нагоре
  ↓ / S  – надолу
  ← / A  – наляво
  → / D  – надясно
  R       – рестарт (след край на играта)
  Q       – изход


ПРАВИЛА
-------
  • Изяжте всички бели точки, за да спечелите.
  • Голямата жълта точка (О) активира Power Mode за ~3 сек.
  • В Power Mode можете да изядете духовете (+200 точки).
  • Докосването от дух = -1 живот. При 0 живота = Game Over.
  • При Game Over или победа се показва класацията в конзолата.


ТОЧКОВА СИСТЕМА
---------------
  Малка точка   →  10 точки
  Power-Up      →  50 точки
  Изяден дух    → 200 точки


ОПИСАНИЕ НА ФАЙЛОВЕТЕ (спрямо изискванията)
--------------------------------------------
  models.py
    - Клас PacManGame с конструктор __init__(self, ...)
    - Методи: _create_maze, _count_dots, move_pacman,
              move_ghosts, _check_ghost_collision,
              get_sorted_scores, get_status, reset
    - get_sorted_scores() демонстрира сортиране (sorted())

  main.py
    - Импортира и инстанцира PacManGame
    - Pygame игрови цикъл с цикли и условни изрази
    - Вика всички методи на обекта
